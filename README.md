
# Breast Cancer Classification using Neural Networks

## Overview
This project uses a Neural Network built with TensorFlow and Keras to classify breast cancer tumors as Benign or Malignant. The model is trained using the Breast Cancer Wisconsin Dataset from Scikit-learn.

## Features
- Breast Cancer Classification
- Data Preprocessing
- Feature Standardization
- Neural Network using TensorFlow & Keras
- Model Training and Evaluation
- Prediction of Tumor Type

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
The project uses the Breast Cancer Wisconsin Dataset available in the Scikit-learn library.

## Project Structure
```
Breast_Cancer_Classification_with_Neural_Network.ipynb
README.md
requirements.txt
```

## Results
The model classifies tumors into:
- Benign
- Malignant
